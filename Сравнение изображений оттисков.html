2. Автоматическое сравнение изображений

  • ImageMagick (командная строка):

  • OpenCV (Python) – Позволяет сравнивать изображения алгоритмами обработки изображений.

3. Онлайн-сервисы

Если различия минимальны (например, изменения формы из-за угла наклона, освещения), можно использовать OpenCV + Feature Matching (поиск совпадающих ключевых точек), а также нейросетевые методы, например, через SIFT, ORB, SURF.

  2. Нажмите "New repository" (Создать репозиторий).
  3. Введите название, например, "stamp-comparison-app".
  4. Оставьте репозиторий публичным и нажмите "Create repository".

📌 Шаг 2. Копируем репозиторий на компьютер

  1. Откройте Терминал (или Командную строку).
  2. Выполните команду (замените your-username на свой GitHub-логин):
    bash
    git clone https://github.com/your-username/stamp-comparison-app.git
cd stamp-comparison-app
  3. Теперь у вас есть папка "stamp-comparison-app" с пустым репозиторием.

📌 Шаг 3. Добавляем код FastAPI

  1. Скачайте файл с кодом main.py и сохраните его в папку stamp-comparison-app.
  2. Создайте файл requirements.txt и вставьте в него:
    nginx
    fastapi
uvicorn
opencv-python
numpy

📌 Шаг 4. Загружаем код в GitHub

  1. В терминале (находясь в папке stamp-comparison-app) выполните:
    bash
    git add .
git commit -m "Initial commit"
git push origin main
  2. Теперь код появился в вашем репозитории на GitHub! 🎉

📌 Шаг 5. Разворачиваем сервер на Render (бесплатно)

  1. Перейдите на Render и войдите через GitHub.
  2. Нажмите "New Web Service".
  3. Выберите свой репозиторий stamp-comparison-app.
  4. В настройках укажите:
    • Runtime: Python
    • Start Command:
      bash
      uvicorn main:app --host 0.0.0.0 --port 8000
    • Free Plan (бесплатный тариф).
  5. Нажмите Deploy – сервис автоматически развернет FastAPI! 🚀

📌 Шаг 6. Проверяем сервер

После деплоя Render выдаст вам ссылку (например, https://stamp-app.onrender.com).
Проверьте в браузере:

  • Откройте https://stamp-app.onrender.com/docs
  • Вы увидите интерфейс API и сможете загружать файлы.

📌 Почему здесь нужен Git и локальная установка?

1️⃣ Git нужен для управления кодом

  • Чтобы ваш код FastAPI был загружен в GitHub, его сначала нужно закоммитить локально.
  • После этого Render (или другой хостинг для бэкенда) сможет взять код из GitHub и развернуть сервер.

2️⃣ Бэкенд нельзя просто закинуть на Vercel

  • Vercel не поддерживает Python и FastAPI – только JavaScript (Node.js).
  • Нам нужно развернуть FastAPI-сервер на Render (или другом сервисе).
  • Render возьмет код из GitHub, поэтому мы сначала заливаем его туда через Git.

3️⃣ Локальная установка Git – это базовый инструмент

  • В будущем, если захотите изменять код, улучшать приложение, добавлять новые функции, вам все равно придется работать с Git.
  • Это стандартная практика для всех веб-разработчиков.

from fastapi import FastAPI, File, UploadFile
import cv2
import numpy as np
import shutil
from pathlib import Path
from fastapi.responses import FileResponse
import uvicorn
import os

app = FastAPI()
UPLOAD_DIR = Path("uploads")
UPLOAD_DIR.mkdir(exist_ok=True)

def process_images(image1_path: str, image2_path: str):
img1 = cv2.imread(image1_path, cv2.IMREAD_GRAYSCALE)
img2 = cv2.imread(image2_path, cv2.IMREAD_GRAYSCALE)
if img1.shape != img2.shape:
img2 = cv2.resize(img2, (img1.shape[1], img1.shape[0]))
diff = cv2.absdiff(img1, img2)
_, diff_thresh = cv2.threshold(diff, 30, 255, cv2.THRESH_BINARY)
overlay = cv2.addWeighted(img1, 0.5, img2, 0.5, 0)
output_path = UPLOAD_DIR / "comparison_result.png"
cv2.imwrite(str(output_path), overlay)
return str(output_path)

@app.post("/compare")
async def compare_images(file1: UploadFile = File(...), file2: UploadFile = File(...)):
file1_path = UPLOAD_DIR / file1.filename
file2_path = UPLOAD_DIR / file2.filename
with file1_path.open("wb") as buffer:
shutil.copyfileobj(file1.file, buffer)
with file2_path.open("wb") as buffer:
shutil.copyfileobj(file2.file, buffer)
result_path = process_images(str(file1_path), str(file2_path))
return FileResponse(result_path, media_type="image/png", filename="comparison_result.png")

if __name__ == "__main__":
uvicorn.run(app, host="0.0.0.0", port=8000)
